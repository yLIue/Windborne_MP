# CiscoNote

## 特殊符号标记

用户模式下输入标记为>
特权模式下输入标记为#
全局模式下输入标记为!
其他模式下输入标记为-
cmd模式下输入标记为^
可自定义的标记为<>

## 通用设备

| 描述          | 指令                  | 快捷键       |
| ------------- | --------------------- | ------------ |
| 帮助          | ?                     |              |
| 特权模式>     | en                    |              |
| 全局配置模式# | conf t                |              |
| 接口模式!     | in `<接口>`           |              |
| 批量接口模式! | in range `<批量接口>` |              |
| 主机名!       | ho `<name_id>`        |              |
| 特权密码明文! | ena pass `<密码>`     |              |
| 特权密码密文! | ena s `<密码>`        |              |
| VLAN配置模式- | vlan `<vlan号>`       |              |
| 返回          | ex                    |              |
| 返回特权模式  | end                   | ctrl+c       |
| 退出解析      |                       | ctrl+shift+6 |
